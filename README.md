# typescript-basics

## Setup

initialise a node repo:

```bash
yarn init
```

add typescript dependencies as dev:

```bash
yarn add typescript -D
```

```bash
yarn add ts-node -D
```

initialise typeScript

```bash
npx tsc --init
```

test our our script in the console:

```bash
npx run index.ts
```
